# Станция за печат — настройка

Целта: всяко сканиране да отпечатва етикета автоматично, без диалог и без щракане.

## Еднократна настройка (5 минути)

**1. Задай принтера по подразбиране**
Свържи принтера за етикети и го направи „по подразбиране":
- **Mac:** System Settings → Printers & Scanners → избери принтера → „Set default".
- **Windows:** Settings → Bluetooth & devices → Printers → принтерът → „Set as default".

**2. Сложи стартера на работния плот (Desktop)**
- **Mac:** копирай файла **`Доставки-Хубенов-Печат.command`** на Desktop.
- **Windows:** копирай **`Hubenov-Print-Station.bat`** на Desktop.

**3. Първо стартиране**
- **Mac:** десен бутон върху файла → **Open** → потвърди веднъж (заради защитата на macOS). После просто двойно щракване.
- **Windows:** двойно щракване (ако се появи „Windows protected your PC" → More info → Run anyway).

## Всеки ден

1. Двойно щракване върху стартера → отваря се специален прозорец на Chrome.
2. Влез веднъж (остава логнат за следващите дни).
3. Сканирай — етикетът излиза автоматично на принтера. Готово.

> Стартерът отваря отделен Chrome профил (`HubenovStation`), за да не пречи на личния браузър и да остане логнат. Затваряш прозореца в края на деня.

---

### English (quick)
1. Set the label printer as the **default** printer (one time).
2. Put the launcher on the Desktop (`.command` on Mac, `.bat` on Windows).
3. First run: Mac → right-click → Open; Windows → More info → Run anyway.
4. Daily: double-click → log in once → scan. Labels print silently to the default printer (Chrome `--kiosk-printing`).
